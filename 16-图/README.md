# 图

- 图是网络结构的抽象模型，是一组由边连接的节点
- 图可以表示任何二元关系，比如道路、航班......
- JS 中没有图，但是可以用 Object 和 Array 构建图
- 图的表示法：邻接矩阵、邻接表、关联矩阵

## 图的常用操作

- 深度优先遍历
  - 尽可能深的搜索图的分支
  - 算法口诀
    - 访问根结点
    - 对根结点的没访问过的相邻节点挨个进行深度优先遍历。
- 广度优先遍历
  - 先访问离根结点最近的节点
  - 算法口诀
    - 新建一个队列，把根结点入队
    - 把队头出队并访问
    - 把队头的没访问过的相邻节点入队
    - 重复第二、三步，直到队列为空
